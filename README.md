# Informações do projeto

## Tecnologias usadas

Front End: React + TypeScript + Material UI  
Back End: Java + Spring Boot  
Banco de dados: PostgreSQL

## Hospedagem

Todas as instâncias do projeto foram hospedadas na nuvem.

Banco de dados: Aiven  
Back End: Render  
Front End: Vercel

Link da aplicação: https://desafio-visual-s.vercel.app

O que pensei então foi deixar os botões em coluna e permitir o overflow-x, para que a pessoa consiga ter visibilidade total da tabela.
